# Password Generator

A simple web app to generate strong passwords easily.

## Features
- Choose password length and character types (numbers, symbols)
- Save passwords in the browser using Local Storage

## Tech
HTML, CSS, JavaScript

## How to Use
1. Enter desired password length
2. Select character types
3. Click "Generate Password"
4. Copy the generated password
