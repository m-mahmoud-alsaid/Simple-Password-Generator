# Password Generator

A simple web app to generate secure passwords with options for numbers and special characters.  
It also keeps the **last 10 generated passwords**.

---

## Features

- Set password length (1–32, default 10)  
- Include numbers and/or special characters  
- Displays the generated password instantly  
- Shows the last 10 passwords  
- Saves passwords in `localStorage`  

---

## Usage

1. Enter desired password length (optional).  
2. Check options for numbers and special characters.  
3. Click **Generate Password**.  
4. See the password and the latest 10 generated passwords below.  

---

## Technical

- Built with **HTML, CSS, and JavaScript**  
- Uses `localStorage` to save passwords  
- Escapes HTML for safe display
